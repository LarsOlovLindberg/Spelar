Live – Status & felsökning

Syfte: kunna avgöra om systemet är “levande” utan att gissa. Den här sidan visar snapshots som portalen läser från /web/data/.

Snabb verifikation (1 minut)

  • Titta på Live-check överst i portalen: filernas ålder ska minska efter “Ladda om”.
  • Klicka live_status.json och kontrollera att timestamp/fält ser rimliga ut.
  • Om executed_trades.csv är tom är det inte nödvändigtvis fel — men den ska uppdateras när något faktiskt exekveras.

Vanliga felbilder

  • LIVE blir rött/gammalt: snapshots uppdateras inte (VPS-agent stoppad / sync / FTP).
  • LIVE ok men data ser fel ut: pipeline/logik producerar fel values (inte UI).
  • Filer saknas: de finns inte i /web/data/ på webhotellet.

Heartbeat

Running

JSON: /web/data/live_status.json (snabb status, senaste loop, ev. fel).

Senaste trades

CSV: /web/data/executed_trades.csv (när något faktiskt exekveras ska nya rader synas här).

Tips

  • Om du vill se Polymarket-uppkoppling separat: gå till Polymarket status i menyn.
  • Om du vill se signal → fill: gå till Kraken futures + Execution log.
